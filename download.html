Part of the project development relies on Bootstrap, which should be first introduced in the introduction project to avoid conflicts
Fonts should be copied in the fonts folder of the CSS folder (CSS /fonts)

CSS files are compiled and compressed as a whole

Download compiled and make it immediately available and add to your project.The following files are included in the download package

  • unis.css
  • fonts
    • unis.eot
    • unis.ttf
    • unis.woff
    • unis.svg
    • Avenir-fontlib
      • AvenirNext-Bold
      • AvenirNext-DemiBold
      • AvenirNext-Medium
      • AvenirNext-Regular
  • Bootstrap v4

Example:

download (fontsCss.zip) download (unis.css) download (bootstrap.css) calendars (calendars.css)
